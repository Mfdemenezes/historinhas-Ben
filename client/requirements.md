## Packages
framer-motion | Essential for the page flipping animations and smooth transitions
react-use | Useful hooks for various UI interactions
canvas-confetti | For celebration effects when finishing a book

## Notes
The application requires a "book" feel. 
We will use Framer Motion to simulate the page turning effect as it offers more control than specialized libraries often do in React 18+.
Images will be handled via URL inputs for the MVP as requested.
The color palette will be playful (pastels + vibrant accents).
Fonts will be 'Comic Neue' for a handwritten feel and 'Outfit' for UI elements.
